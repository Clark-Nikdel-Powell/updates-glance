Shows the number of installed themes and plugins and a count of pending updates for each.

== Description ==

This is a simple plugin that will display the number of installed plugins and themes in the At A Glance dashboard widget.
It also shows the number of pending plugins and themes updates.

Use the filter `update_glance_items` if you wish to remove the plugin or theme item from the At A Glance dashboard widget.

== Installation ==

1. Upload the `updates-glance` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
